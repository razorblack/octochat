OctoChat

About Us

OctoChat is an opensource chat application for GitHub users.
It is a web application to easily connect people on the GitHub for better communication/interaction among them.

Get Started

Technology & Tools

OctoChat is build using the following technology and tools available.

Microsoft Azure

Microsoft Azure, often referred to as Azure, is a cloud computing service operated by Microsoft for application management via Microsoft-managed data centers.

Git & GitHub

Git is a free and open source distributed version control system. GitHub, Inc. is a provider of Internet hosting for software development and version control using Git.

HTML, CSS & Bootstrap

HTML is the standard markup language for documents designed to be displayed in a web browser. CSS is a style sheet language used for describing the presentation of a document written in a markup language. Bootstrap is a free and open-source CSS framework.

JavaScript, Node & React

JavaScript is the programming language of the Web. Node.js is a JavaScript runtime built on Chrome's V8 JavaScript engine. React is a free and open-source front-end JavaScript library for building user interfaces based on UI components

Features

Octochat is loaded with the following features to increase your overall experience. LogIn/SignUp to get started.

Simple LogIn/SignUp

Simple and fast LogIn/SignUp directly using your GitHub account.

Cloud Storage & Backup

Complete data is stored and backed up on Microsoft Azure cloud environment.

Data Security

Data remains secured using security protocols and techniques of Azure cloud.

Notification

Get notified everytime someone messages you or connects with you.

GitHub Community

Find and connect with developers on GitHub directly without any hassle of finding their contacts.

Group Chats

Communicate with multiple people at the same time or instantly chat to everyone in your team.
